# News Explorer Backend

API REST para la aplicación News Explorer.

## Dominios

- Frontend: https://news-explorer-g.mooo.com
- API: https://api.news-explorer-g.mooo.com

## Endpoints

- `POST /signup` — registro de usuario
- `POST /signin` — inicio de sesión
- `GET /users/me` — obtener usuario actual
- `GET /articles` — obtener artículos guardados
- `POST /articles` — guardar artículo
- `DELETE /articles/:articleId` — eliminar artículo

## Tecnologías

- Node.js
- Express
- MongoDB
- JWT
- Mongoose
- Nginx
- Google Cloud
